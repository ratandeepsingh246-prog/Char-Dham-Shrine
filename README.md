# Char-Dham-Shrine
Power BI dashboard analyzing Char Dham shrine pilgrim trends across multiple years. It presents total visitors, minimum and maximum footfall, year-wise growth rates, CAGR by shrine, and comparative analysis of Badrinath, Kedarnath, Gangotri, Yamunotri, and Hemkund Sahib to understand tourism patterns, demand fluctuation through interactive visuals.

# Char Dham Pilgrim Analysis Dashboard

This project is a Power BI dashboard focused on analyzing pilgrim trends across the Char Dham shrines.

## Project Overview
The dashboard provides a comprehensive view of pilgrim footfall across Badrinath, Kedarnath, Gangotri, Yamunotri, and Hemkund Sahib over multiple years. It highlights total visitors, minimum and maximum yearly visitors, and long-term trends to understand tourism patterns.

## Key Insights
- Total pilgrim count across all shrines  
- Shrine-wise visitor distribution  
- Year-wise pilgrim trends and comparisons  
- Growth rate analysis by year and shrine  
- CAGR analysis to measure long-term growth  
- Identification of peak and low tourism periods  

## Tools Used
- Power BI  
- Char Dham Pilgrim Dataset  

## Purpose
The purpose of this project is to analyze religious tourism trends, identify growth patterns, and demonstrate data visualization and analytical skills using Power BI.
