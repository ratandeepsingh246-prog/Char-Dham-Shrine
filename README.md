# Char-Dham-Shrine
Power BI dashboard analyzing Char Dham shrine pilgrim trends across multiple years. It presents total visitors, minimum and maximum footfall, year-wise growth rates, CAGR by shrine, and comparative analysis of Badrinath, Kedarnath, Gangotri, Yamunotri, and Hemkund Sahib to understand tourism patterns, demand fluctuation through interactive visuals.
